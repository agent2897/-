# Lab 3 Python Project

Ця директорія містить приклад реалізації UML-діаграми класів для системи тимчасового утримання.

## Файли

- `models.py` — моделі та класи, що відповідають UML-діаграмі
- `repository.py` — шаблон Repository для зберігання сутностей
- `demo.py` — приклад використання класів

## Як запустити

1. Відкрийте термінал у `Лабораторна робота 3/lab3_project`
2. Запустіть `python demo.py`

Цей приклад показує, як створити затриманого, обліковий запис утримання, чергового, медичного працівника, подію, камеру та документ.
