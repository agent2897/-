# Lab 2 Python Project

Ця директорія містить реалізацію трьох Use Case для системи ІТУ:
- UC-1: реєстрація затриманого
- UC-2: контроль строків утримання
- UC-3: оформлення звільнення

## Запуск

1. Перейдіть у директорію `Лабораторна робота 2/lab2_project`
2. Запустіть `python demo.py`

Файл `demo.py` демонструє основні сценарії роботи системи згідно з Use Case описом.
