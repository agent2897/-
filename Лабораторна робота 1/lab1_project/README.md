# Lab 1 Python Project

Ця директорія містить приклад реалізації вимог та структури системи для лабораторної роботи 1.

## Мета

Реалізувати базові функції автоматизації обліку затриманих у ІТУ:
- реєстрація нового затриманого;
- контроль строків утримання;
- розміщення по камерах;
- формування документів.

## Запуск

1. Перейдіть у директорію `Лабораторна робота 1/lab1_project`
2. Запустіть `python demo.py`

Файл `demo.py` демонструє реєстрацію затриманого, обчислення строку утримання та розміщення у камері.
